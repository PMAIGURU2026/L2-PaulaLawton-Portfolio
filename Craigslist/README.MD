# 🏘️ Craigslist Housing Equity Project
### Pursuit AI Fellowship — Level 2

## What It Does
A tool that analyzes Craigslist housing listings to surface 
patterns related to housing equity, affordability, and access.

## Why We Built It
Housing discrimination is often hidden in plain sight.
This tool makes those patterns visible through data.
