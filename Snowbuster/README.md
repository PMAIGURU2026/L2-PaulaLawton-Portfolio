# ❄️ Snow Busters
### Pursuit AI Fellowship — Level 1 | Hackathon Project

## What It Does
A two-sided dispatch app connecting customers who need snow 
cleared (driveways, walkways, car dig-outs) with workers 
ready to take the job — like TaskRabbit but only for snow.

## Why We Built It
During storms, vulnerable neighbors — elderly residents and 
people with disabilities — can't clear snow safely and have 
no fast way to find help. Snow Busters fixes that.

## Tech Stack
- Frontend: HTML, CSS, JavaScript (responsive + mobile)
- Backend: Node.js + Express
- Database: PostgreSQL
- Status Engine: Requested → Accepted → Completed
- Deploy: Render.com
