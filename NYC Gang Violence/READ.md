# 🗽 NYC Gang Violence Resource Mapper
### Pursuit AI Fellowship — Level 2 | Mini Hackathon

## What It Does
An independent data research tool mapping gang violence 
hotspots across NYC alongside available community resources — 
exposing "resource deserts" where support is most needed 
but least available.

## Why We Built It
Data about violence is everywhere. Data about resources is 
scattered. This tool puts both on the same map so communities 
and policymakers can see the gaps clearly.

## Tech Stack
- Python (data research + analysis)
- Pandas (data cleaning + processing)
- Mapping/visualization tools
- Data sources: NYC Open Data, public crime datasets
- PRD: Full product requirements document written independently
