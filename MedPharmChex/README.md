# MedPharmChex

## Summary
MedPharmChex is a sophisticated project designed to streamline and enhance pharmaceutical management, providing tools and solutions that facilitate better pharmacy practice and patient care.

## Tech Stack
- **Frontend**: React.js
- **Backend**: Node.js, Express
- **Database**: MongoDB
