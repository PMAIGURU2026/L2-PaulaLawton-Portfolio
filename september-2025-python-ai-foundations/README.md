## Description

This project covers the foundational concepts introduced at the start of the Pursuit AI Fellowship Level 2 program. It includes Python refreshers, core data structures, and an introduction to the AI/ML landscape. The goal was to build a strong programming baseline before diving into applied machine learning.

## Topics Covered

- Python fundamentals review (functions, classes, list comprehensions, generators)
- Working with libraries: `NumPy` and `Pandas` for data manipulation
- Introduction to AI concepts: supervised vs. unsupervised learning, training/test splits
- Setting up development environments with `conda`/`venv` and Jupyter Notebooks
- Version control best practices with Git

## Project Highlights

- **Python Exercises:** A collection of Python scripts demonstrating core language features.
- **NumPy Notebook:** Jupyter notebook exploring array operations and broadcasting.
- **Intro to ML Notebook:** Walkthrough of the ML workflow using a sample dataset (Iris), including loading data, feature exploration, and a simple classifier.

## Skills Demonstrated

- Clean, readable Python code following PEP 8 conventions
- Data loading, cleaning, and inspection with Pandas
- Running and documenting experiments in Jupyter Notebooks

## How to Run

```bash
# Install dependencies
pip install numpy pandas scikit-learn jupyter matplotlib

# Launch notebooks
jupyter notebook
```

## Key Takeaways

Starting the fellowship reinforced that a solid Python foundation is essential before tackling AI/ML problems. Learning to frame business questions as machine learning tasks was the most valuable conceptual shift of the month.
